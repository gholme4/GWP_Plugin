GWP_Plugin

GWP_Plugin Class that implements common functions of a WordPress plugin in an effort reduce the amount of functions written in the global namespace.

Requires PHP 5.3.0 and above

is_mobile() : boolean
static

Returns true if user is on a mobile device and false if user is on a desktop device.

Usage
